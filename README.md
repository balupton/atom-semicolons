<!-- DESCRIPTION/ -->

Atom package for removing semicolons in JavaScript files

<!-- /DESCRIPTION -->


## Install

``` bash
apm install atom-semicolons
```


## Tips

Add the following to your keymap file to remove unnecessary semicolons on save:

``` coffeescript
'.editor':
	'cmd-s': 'semicolons:remove'
```
